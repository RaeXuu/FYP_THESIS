## 数据驱动图表（matplotlib）制作说明

### 可直接用现有数据出图的

**Fig 5.2 — 消融实验分组柱形图**（Table 5.4 数据已有）

```python
import matplotlib.pyplot as plt
import numpy as np

configs = ['A: Baseline', 'B: +Wider', 'C: +CoordAtt', 'D: +Residual']
m_scores = [0.8851, 0.8896, 0.8869, 0.8912]
se       = [0.9654, 0.9595, 0.9383, 0.9797]
sp       = [0.8049, 0.8198, 0.8355, 0.8027]

x = np.arange(len(configs))
w = 0.25
fig, ax = plt.subplots(figsize=(9, 5))
ax.bar(x - w, m_scores, w, label='M-Score')
ax.bar(x,     se,       w, label='Sensitivity')
ax.bar(x + w, sp,       w, label='Specificity')
ax.set_xticks(x); ax.set_xticklabels(configs, rotation=15, ha='right')
ax.set_ylim(0.75, 1.0); ax.set_ylabel('Score')
ax.legend(); ax.grid(axis='y', alpha=0.4)
plt.tight_layout()
plt.savefig('fig5_2_ablation.pdf', dpi=300)
```

## 代码生成（PyTorch）

### torchviz — 自动追踪计算图

自动展示 Coordinate Attention 内部结构和残差连接（分叉后汇合的箭头）。

```python
import torch
from torchviz import make_dot

model = YourCNNModel()
x = torch.randn(1, 3, 224, 224)  # 根据输入尺寸调整
y = model(x)

dot = make_dot(y, params=dict(model.named_parameters()))
dot.render("model_arch", format="png")
```

> 缺点：图比较底层，节点是操作而非层，适合调试但不适合论文插图。

### 其他

## draw.io 画法建议

针对 CNN + Coordinate Attention + 残差连接：

- 用矩形块表示卷积层、BN、激活函数
- Coordinate Attention 模块单独用不同颜色的子图，内部结构：
  - H/W 方向 Average Pool → Concat → FC → Split → Sigmoid
- 残差连接用**跳跃箭头**绕过主路径后相加
